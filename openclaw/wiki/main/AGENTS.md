# Memory Wiki - Agent Guide

This vault contains compiled knowledge with provenance, claims, and evidence.

## Structure
- `entities/` — People, systems, projects, tools
- `concepts/` — Ideas, patterns, policies
- `sources/` — Imported raw material
- `syntheses/` — Compiled summaries
- `reports/` — Dashboards and health checks

## Conventions
- Every claim needs evidence and confidence score
- Use `wiki_apply` for mutations, don't hand-edit managed blocks
- Run `wiki lint` after meaningful changes
- Run `wiki compile` to refresh indexes and dashboards
