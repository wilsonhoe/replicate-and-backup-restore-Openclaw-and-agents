## 🧮 Built-In Calculations

All essential trading formulas included:

```
P&L (Long) = (Exit Price - Entry Price) × Quantity - Fees
P&L (Short) = (Entry Price - Exit Price) × Quantity - Fees
P&L % = (P&L / Position Size) × 100
Win Rate = (Winning Trades / Total Trades) × 100
Profit Factor = Gross Profit / Gross Loss
Expectancy = (Win Rate × Avg Win) - (Loss Rate × Avg Loss)
Risk/Reward = Avg Win / Avg Loss
Position Size = (Account Size × Risk %) / (Entry - Stop Loss)
```
