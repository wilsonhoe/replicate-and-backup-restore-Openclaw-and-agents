## 🚀 How to Import the Template into Notion

### Method 1: Direct Import (Recommended)

1. Open Notion and create a new page
2. Click **Import** in the left sidebar
3. Select **HTML** as the import format
4. Choose the `crypto-trading-journal.html` file
5. Wait for import to complete
6. Review and customize as needed

### Method 2: Copy & Paste

1. Open `crypto-trading-journal.html` in a web browser
2. Select all content (Ctrl/Cmd + A)
3. Copy (Ctrl/Cmd + C)
4. Create a new page in Notion
5. Paste (Ctrl/Cmd + V)
6. Format adjustments may be needed

### Method 3: Drag & Drop

1. Open Notion to an empty page
2. Drag the HTML file directly into Notion
3. Notion will automatically parse and import the content

## 🧮 Built-In Formulas

All formulas are documented in the template for easy Notion formula setup:

```
// P&L Calculations
Long: (Exit Price - Entry Price) × Quantity - Entry Fees - Exit Fees
Short: (Entry Price - Exit Price) × Quantity - Entry Fees - Exit Fees
P&L %: (P&L / Position Size) × 100

// Performance Metrics
Win Rate: (Winning Trades / Total Trades) × 100
Profit Factor: Gross Profit / Gross Loss
Expectancy: (Win Rate × Avg Win) - (Loss Rate × Avg Loss)
Risk/Reward: Avg Win / Avg Loss

// Risk Management
Position Size: (Account Size × Risk %) / (Entry Price - Stop Loss)
Risk Amount: Account Size × Risk %
```
