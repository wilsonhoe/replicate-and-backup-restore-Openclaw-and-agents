### New Media Tools
```javascript
// Video generation
video_generate({
  prompt: "Product showcase animation",
  durationSeconds: 10
})

// Music generation  
music_generate({
  prompt: "Upbeat corporate background music",
  instrumental: true
})
```

### CLI Timeout Workaround
If `openclaw cron` commands timeout/SIGKILL, use direct file reads:
- Cron config: `~/.openclaw/cron/jobs.json`
- Session logs: `~/.openclaw/agents/{agent}/sessions/{id}.jsonl`
