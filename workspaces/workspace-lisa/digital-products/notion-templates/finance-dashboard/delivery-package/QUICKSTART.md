# Quick Start Guide - Notion Finance Dashboard

**Setup Time:** 10-15 minutes

---

## Step 1: Duplicate the Template (1 minute)

1. Click the duplicate link you received
2. Select your Notion workspace
3. Click "Duplicate"
4. Wait for it to copy (5-10 seconds)

✅ Done! The template is now in your workspace.

---

## Step 2: Add Your First Transactions (5 minutes)

Go to the **💸 Transactions** database and add 5-10 entries:

### For Income:
1. Click **New** (top right)
2. Name: e.g., "Website Project - Client ABC"
3. Amount: e.g., `2500` (positive number)
4. Category: Select **Freelance** or **Income**
5. Date: Select the date
6. Type: Select **Income**
7. Status: Select **Completed**
8. Source: Select **Bank**, **PayPal**, etc.

### For Expenses:
1. Click **New**
2. Name: e.g., "Office Supplies"
3. Amount: e.g., `-150` (negative for expenses)
4. Category: Select **Expense-Business** or similar
5. Date: Select the date
6. Type: Select **Expense**
7. Status: Select **Completed**
8. Source: Select **Credit Card**, etc.

**Pro Tip:** Add at least 5 transactions to see the dashboard come alive!

---

## Step 3: Set Up Your Budget (3 minutes)

Go to the **💰 Budget Planner** database:

1. Click **New**
2. Category: e.g., "Marketing & Ads"
3. Monthly Limit: e.g., `500`
4. Current Spent: Enter what you've spent this month (e.g., `275`)
5. Remaining: Automatically calculated!
6. Status: Automatically shows 🟢, 🟡, or 🔴

**Repeat for each budget category:**
- Marketing & Ads
- Software & Tools
- Office Supplies
- Travel
- Professional Development
- etc.

**Pro Tip:** Start with 5-7 categories. You can always add more later!

---

## Step 4: Add Outstanding Invoices (3 minutes)

Go to the **📋 Invoice Tracker** database:

1. Click **New**
2. Invoice #: e.g., "INV-2026-001"
3. Client: e.g., "ABC Corporation"
4. Amount: e.g., `3500`
5. Date Sent: Select the date
6. Due Date: Select the due date
7. Status: Select **Sent** (or **Overdue** if past due)
8. Notes: Add any details

**Pro Tip:** Use this to track who owes you money. Check it weekly!

---

## Step 5: Set a Financial Goal (2 minutes)

Go to the **🎯 Financial Goals** database:

1. Click **New**
2. Goal: e.g., "Emergency Fund - $10K"
3. Target Amount: e.g., `10000`
4. Current Amount: e.g., `6500`
5. Deadline: Select your target date
6. Priority: Select **High**, **Medium**, or **Low**
7. Status: Select **In Progress**
8. Progress %: Automatically calculated!

**Pro Tip:** Start with one important goal. Add more as you go!

---

## Step 6: Explore the Dashboard (1 minute)

Scroll through the page and notice:

- ✅ Transactions show your cash flow
- ✅ Budget shows where you stand
- ✅ Invoices show who owes you
- ✅ Goals show your progress

**This is YOUR system now.** Customize it however you want!

## Customization Tips

### Change Categories:
1. Click any category dropdown
2. Click "Edit" at the bottom
3. Add, remove, or rename categories

### Add New Properties:
1. Click **+** at the end of column headers
2. Select property type (Text, Number, Date, etc.)
3. Name your new property

### Create New Views:
1. Click the database name
2. Click **+ Add a view**
3. Choose: Table, Board, Calendar, Timeline, etc.
4. Set your filters and sorts

### Change Formulas:
1. Click the formula property
2. Click **Edit formula**
3. Modify as needed (see FORMULAS.md for reference)

## Troubleshooting

**Q: I can't see the formulas working**
A: Make sure you've entered numbers in the required fields. Formulas calculate automatically.

**Q: My budget isn't updating**
A: The Budget Planner doesn't auto-link to Transactions (yet). Manually update "Current Spent" or set up relations.

**Q: The duplicate button doesn't work**
A: Make sure you're logged into Notion. Try opening in a new browser window.

**Q: Can I delete sample data?**
A: Yes! Select all rows and delete. Or keep it as reference.

**Q: Can I share this with my team?**
A: Yes! Duplicate to a team workspace or share the page.
