**Price:** $29 | **Target:** Content Creators, Social Media Managers

## DATABASE SCHEMA

### Database 1: Content Calendar (Main)
**Type:** Calendar view (primary) + Table

**Properties:**
| Property | Type | Options |
|----------|------|---------|
| Content Title | Title | - |
| Platform | Select | Instagram, TikTok, YouTube, Twitter/X, LinkedIn, Facebook, Blog, Email |
| Content Type | Select | Reel, Carousel, Story, Post, Video, Thread, Article |
| Status | Select | Idea, Planned, In Progress, Scheduled, Published, Archived |
| Publish Date | Date | - |
| Time | Text | e.g., "9:00 AM" |
| Hook/First Line | Text | - |
| Caption/Script | Text | - |
| CTA | Text | - |
| Hashtags | Multi-select | #entrepreneur, #contentcreator, etc. |
| Media | Files | - |
| URL | URL | Published link |
| Views | Number | - |
| Engagement | Number | - |
| Performance | Formula | Auto-calculate engagement rate |

**Views:**
1. Calendar (default) - by Publish Date
2. Board by Status
3. Board by Platform
4. This Week filter
5. Published (Performance tracking)

### Database 2: Content Ideas
**Type:** Gallery

**Properties:**
| Property | Type | Options |
|----------|------|---------|
| Idea | Title | - |
| Category | Select | Education, Inspiration, Promotion, Behind-the-scenes, Engagement |
| Source | Select | Brainstorm, Trending, Audience Q, Performance Analysis |
| Priority | Select | High, Medium, Low |
| Status | Select | New, In Review, Approved, Rejected |
| Potential Hook | Text | - |
| Notes | Text | - |

### Database 3: Hashtag Library
**Type:** Table

**Properties:**
| Property | Type | Options |
|----------|------|---------|
| Hashtag | Title | - |
| Category | Select | Business, Marketing, Productivity, Finance, Lifestyle, Tech |
| Platform | Multi-select | Instagram, TikTok, LinkedIn, etc. |
| Post Count | Number | If tracked |
| Best Performing | Checkbox | - |
