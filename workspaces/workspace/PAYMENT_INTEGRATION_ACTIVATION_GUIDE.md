## 🚨 TROUBLESHOOTING GUIDE

### **Common Issues & Solutions**

**Issue**: "Can't upload PDF file"
- **Solution**: Check file size (<5GB limit), try compressing if needed, ensure not corrupted

**Issue**: "Payment not processing"  
- **Solution**: Verify bank details in Settings → Payouts, check for verification requirements, contact Gumroad support

**Issue**: "Customers can't download after purchase"
- **Solution**: Check product settings for download limits, verify file was uploaded correctly, resend download link manually if needed

**Issue**: "Need to issue refund"
- **Solution**: Go to Sales → Find transaction → Click "Refund" → Gumroad processes automatically

**Issue**: "Not seeing sales in dashboard"
- **Solution**: Check timezone settings, verify payment completed successfully, look under "Recent Sales"
