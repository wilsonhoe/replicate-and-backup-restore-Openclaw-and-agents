   "Hello, this is [name] from [agency]. I'm calling because I saw your property listing expired. Are you still looking to sell?"

   "I understand your concern about the price. Let me show you some recent comparable sales in this area to help set realistic expectations."

## How to Import This Template

### Step 1: Create Your Workspace
1. Open Notion (free account works fine)
2. Create a new page called "SG Property Pro"
3. Add this markdown content

### Step 2: Create Databases
1. Type `/database` and select "Database - Inline"
2. Create each database with the schema above
3. Add properties carefully matching the types
4. Add formulas as specified

### Step 3: Import Data
1. Use the sample data tables above
2. Copy each row into your databases
3. Or import via CSV (Notion Pro feature)

### Step 4: Set Up Dashboard
1. Create a new page called "Dashboard"
2. Link to each database using `/link`
3. Add filtered views (e.g., Priority = 🔥 Hot)
