# Door Knocking Mastery System

> Complete Singapore real estate door knocking system for the NEL corridor — Punggol, Serangoon, Sengkang

---

## What Is This?

A **complete execution + training + tracking system** for real estate agents doing door-to-door sales in Singapore HDB estates.

**Not just a CRM.** This system:
- **Tracks** every session with auto-calculated KPIs
- **Trains** you with daily practice scripts  
- **Optimizes** through weekly performance reviews
- **Converts** with structured follow-up workflows

## System Components

### 1. Door Knocking Sessions Database
Track every field session:
- Date, Area, Block, Time Slot
- Doors Knocked, Opened, Conversations
- Leads Captured, Appointments Set
- **Auto-calculated KPIs:** Open Rate %, Conversation Rate %, Lead Rate %

### 2. Leads Database
HDB-specific pipeline:
- Owner/Tenant distinction
- Intent (Sell/Buy/Exploring)
- Timeline and Budget
- Source tracking (auto: Door Knocking)
- Status flow: New → Follow-up → Viewing → Closed/Lost
- Priority scoring (High/Medium/Low)

### 3. Follow-ups Database
Structured conversion system:
- Follow-up date, Method (Call/WhatsApp/Visit)
- Outcome tracking
- Next action defined
- Priority levels
- Status: Pending/Done/Overdue

**Views:**
- Due Today
- Overdue  
- High Priority
- By Method

### 4. Training Sessions Database
Daily skill practice:
- Practice Type: Warm-up / Simulation / Review
- Difficulty: Easy / Medium / Hard
- Scenario Type: Punggol / Serangoon / Objection handling
- Script used
- Score (1-10)
- Feedback and improvement notes

**Workflow:**
1. Warm-up (10 min) — Opening lines
2. Simulation (15 min) — Roleplay scenarios
3. Review (20 min) — What worked/didn't

### 5. Scripts Database
Complete script library:
- Punggol scripts (young families)
- Serangoon scripts (mature owners)
- Objection handling (universal)
- Follow-up templates

**Each script includes:**
- Content
- Effectiveness score
- Usage count
- Area optimization

### 6. Weekly Reviews Database
Optimization loop:
- Total sessions, doors knocked, leads
- Performance by area
- Script effectiveness analysis
- Lessons learned
- Next week plan
