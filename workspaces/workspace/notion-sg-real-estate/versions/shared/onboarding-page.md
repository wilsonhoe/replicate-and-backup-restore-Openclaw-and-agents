## ❓ Need Help?

### Quick Answers

**"I don't see the imported data"**
→ Make sure you created the database properties FIRST (with exact names), then imported CSV.

**"Formulas show errors"**
→ Property names must match EXACTLY. Check for extra spaces or different capitalization.

**"Can I customize the fields?"**
→ Yes! Add your own properties anytime. Don't delete existing ones or formulas break.

**"How do I add a team member?"**
→ Agency edition: Add agent names to the "Assigned Agent" property options in each database.
