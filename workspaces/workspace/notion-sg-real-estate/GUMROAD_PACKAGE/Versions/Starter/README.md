# SG Property Pro - Starter Edition ($9)

## What's Included

Perfect for new agents building their first client database.

### Databases (2)

1. **Leads Database**
   - Contact info (name, phone, email)
   - Source tracking (Door Knocking, Referral, Online, etc.)
   - Property interest (HDB, Condo, EC, Landed)
   - Budget range tracking
   - Status pipeline (New → Contacted → Viewing → Offer → Closed/Lost)
   - Priority levels (Hot/Warm/Cold)
   - Follow-up dates

2. **Properties Database**
   - Property details (address, type, district)
   - NEL zone areas (Punggol, Sengkang, Hougang, Serangoon)
   - Price & PSF calculations
   - Size, bedrooms
   - Status tracking
