## Production Notes

### Screen Recording Setup
- Use clean Notion workspace (no personal data visible)
- 1920x1080 resolution minimum
- Zoom in on important details (CMD+Plus on Mac)
- Smooth mouse movements (use cursor smoothing if available)

### Audio
- Clear voiceover, no background music during demos
- Optional: Light music during intro/outro
- Normalize audio levels

### Editing
- Cut pauses and mistakes
- Speed up repetitive actions (CSV import)
- Use zoom for emphasis on key features
- Add text overlays for key points

### Thumbnail
Text: "Singapore Real Estate CRM"
Subtext: "Built for Notion"
Visual: Screenshot of dashboard with arrow pointing to key feature
