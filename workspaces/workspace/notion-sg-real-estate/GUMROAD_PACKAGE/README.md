## Quick Start (5 Minutes)

### Step 1: Create Your Notion Workspace
- Sign up at [notion.so](https://notion.so) (free plan works perfectly)

### Step 2: Import the Databases
1. Open Notion → Create a new page
2. Type `/import` → Select **CSV**
3. Import each file from the `Templates/` folder:
   - `leads.csv` — Import as "Leads" database
   - `properties.csv` — Import as "Properties" database
   - `deals.csv` — Import as "Deals" database
   - `activities.csv` — Import as "Activities" database
   - `knowledge_base.csv` — Import as "Knowledge Base" database

### Step 3: Set Up Relations (2 minutes)
- Link each Deal to a Lead and Property
- Link Activities to relevant Leads/Deals
- Done — your CRM is live!

### Step 4: Customize
- Add your branding colors
- Replace sample data with your actual leads
- Adjust commission percentages to match your agency

**Full setup guide:** See `Documentation/QUICK_START.md`

## Key Features Explained

### NEL Zone Targeting
North-East Line corridor (Punggol, Sengkang, Hougang, Serangoon) is one of Singapore's hottest property markets. Built-in tags let you:
- Filter leads by NEL interest
- Track NEL property listings
- Focus your marketing on high-growth zones

### OTP Workflow Tracking
Singapore's Option-to-Purchase process, digitized:
- Option Fee paid (1% of price)
- Exercise Date scheduled
- Completion Date countdown
- Never miss a deadline again

### Commission Calculator
Auto-computed fields:
- Gross commission (2% buyer, 1% seller)
- Agency split (typically 30-50%)
- Net commission per deal
- Weighted pipeline value (probability × commission)

### PSF Analysis
Price Per Square Foot tracking for:
- Competitive property comparison
- Market trend analysis
- Client advisory reports
