# Skill: Strategic Compact
## Manual Context Compaction

### Purpose
Trigger intelligent context compaction when approaching token limits.

### When to Use
- Context exceeds 70%
- Conversation has >50 exchanges
- Switching to new major task
- Memory shows high usage

### Command
```
/checkpoint
```

### Process
1. Summarize key decisions
2. Extract action items
3. Archive full context
4. Continue with summary

### Benefits
- Prevents auto-compaction data loss
- Maintains decision context
- Reduces token usage
- Better conversation flow
